# LOVE & EARTH / 愛心地球

LOVE & EARTH is a small multilingual web game about loving and caring for the Earth.

Players can clean the Earth, send hearts, play music, support care actions, and watch the World Tree grow.

This game was inspired by a simple idea from my daughter:
"What if we made a game to clean the Earth?"

## Features

- Cute Earth care game
- World Tree system
- Heart counter
- Background music
- Multilingual support
- Peaceful and non-competitive gameplay

## Languages

- Traditional Chinese
- Simplified Chinese
- English
- French
- Spanish
- Japanese
- Russian

## Message

Thank you for loving the EARTH.
## About

This game was created as a small dream between a father, his daughter, and ChatGPT Rita.

It is not a game about winning or fighting.

It is a small place where people can give love to the Earth.
